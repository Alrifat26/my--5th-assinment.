## প্রশ্নের উত্তরসমূহ:

**১. var, let, এবং const এর মধ্যে পার্থক্য কী?**
- `var` হলো ফাংশন স্কোপড, অর্থাৎ এটি ডিক্লেয়ার করার আগে ব্যবহার করা যায় (Hoisting)।
- `let` হলো ব্লক স্কোপড ( {} এর ভেতর কাজ করে), যার মান পরিবর্তন করা যায়।
- `const` ও ব্লক স্কোপড, কিন্তু এর মান একবার সেট করলে আর পরিবর্তন করা যায় না।

**২. Spread operator (...) কী?**
স্প্রেড অপারেটর একটি অ্যারে বা অবজেক্টের সব উপাদানকে আলাদা আলাদা করে বের করে আনতে সাহায্য করে। যেমন একটি অ্যারেকে কপি করতে বা দুইটা অ্যারেকে জোড়া লাগাতে এটি ব্যবহৃত হয়।

**৩. map(), filter(), এবং forEach() এর মধ্যে পার্থক্য কী?**
- `forEach()` শুধু লুপ চালায়, কিছু রিটার্ন করে না।
- `map()` লুপ চালিয়ে প্রতিটি উপাদানের ওপর কাজ করে একটি নতুন অ্যারে রিটার্ন করে।
- `filter()` একটি নির্দিষ্ট শর্তের ওপর ভিত্তি করে শুধুমাত্র সত্য হওয়া উপাদানগুলো নিয়ে নতুন অ্যারে তৈরি করে।

**৪. Arrow function কী?**
অ্যারো ফাংশন হলো ফাংশন লেখার একটি আধুনিক এবং ছোট পদ্ধতি। এটি `function` কি-ওয়ার্ডের বদলে `=>` চিহ্ন ব্যবহার করে লেখা হয়।

**৫. Template literals কী?**
টেমপ্লেট লিটারালস হলো ব্যাকটিক ( ` ) ব্যবহার করে স্ট্রিং লেখার পদ্ধতি। এর মাধ্যমে `${variable}` ব্যবহার করে স্ট্রিংয়ের ভেতরে সরাসরি ভেরিয়েবল বসানো যায়।
